# msal-vue-demo
This demo shows how to migrate from msal (1.0) to @azure/msal-browser (2.0) which supports OAuth 2.0 Auth Code Flow with PKCE.
The 1.0 version is tagged msal-1, the 2.0 is tagged msal-2.

## Project setup
```
npm install
```
